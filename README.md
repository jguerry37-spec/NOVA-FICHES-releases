# NOVA-FICHES-releases

Dépôt public minimal utilisé uniquement par le mécanisme de vérification de
mise à jour de Nova-Fiches (application interne NOVATLAS). Il contient un
unique fichier, `version.json`, lu de façon anonyme au démarrage de l'application.

Le code source de Nova-Fiches reste privé, dans un autre dépôt.
